## Perception

[1-5] Modificateur pour détecter des informations utiles.

### Intuition

Avantage pour détecter les changements (vue, ouïe, odorat).

### Perception magique

Avantage lorsqu'il s'agit de magie.

### Mentaliste

Avantage pour détecter les intentions des êtres vivants.
