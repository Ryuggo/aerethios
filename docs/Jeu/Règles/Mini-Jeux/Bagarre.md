>Combat simplifié pour les combats non léthaux (amicaux/ de taverne).

- Chaque joueur : **10 PV**, **D4 de dégâts** (Force = ×2 dégâts).

- Deux approches :
	→ **Force** : brutale
	→ **Technique** : précis et stratégique

| Type      | Calcul                      | Dé  |
| --------- | --------------------------- | --- |
| Force     | Force (Athlétisme)          | D20 |
| Technique | Dextérité + Maîtrise Courte | D12 |

- Pour agir, il faut faire un score **> AC** de l’adversaire.

| Duel                   | Effet                                                      |
| ---------------------- | ---------------------------------------------------------- |
| Force vs Force         | Les deux peuvent infliger des dégâts                       |
| Force vs Technique     | La Technique peut esquiver au lieu d’attaquer              |
| Technique vs Technique | Pierre (Attaquer) / Papier (Esquiver) / Ciseaux (Attraper) |
